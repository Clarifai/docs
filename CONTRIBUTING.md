# Contributing to Clarifai Docs

Thanks for helping improve the Clarifai documentation.

## Getting Started

- Clone the repo and run `npm install` then `npm start` to preview locally.
- All public docs live under `docs/`. The sidebar is autogenerated from the folder structure.
- Branch off `main` for your changes and open a PR when ready.

## Page-Specific Guides

Detailed how-to guides for maintaining specific pages:

- [Updating the Cloud Instances page](contributing/updating-cloud-instances.md)

## General Guidelines

- Keep language clear and direct — write for a new user, not an expert.
- Tables should be ordered most-powerful-first (highest GPU memory / CPU count at the top).
- Use `:::info`, `:::tip`, and `:::note` Docusaurus callouts sparingly and purposefully.
- Do not add pages to the sidebar manually unless necessary — autogeneration handles it.
- Never document features or CLI commands that are not yet in the `main` branch of the relevant SDK/CLI repo.
