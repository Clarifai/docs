## Get Started 

You can quickly get started with just a few lines of code or a few simple clicks.

![](/img/api_v_portal.jpg)

Clarifai offers powerful ways to effortlessly integrate AI into your technology stack.

:::tip Clarifai SDK

[The Clarifai SDK](https://docs.clarifai.com/sdk/python-installation) enables seamless integration of advanced computer vision and natural language processing capabilities into your applications, offering tools for tasks like classification, detection, segmentation, summarization, and more — with just a few lines of code. It's provided in Python and Node.js programming languages. 

:::

:::tip Clarifai gRPC API Clients

[The Clarifai gRPC API clients](https://docs.clarifai.com/api-guide/api-overview/api-clients/) are offered in several popular programming languages, making it easy to integrate Clarifai's AI platform capabilities into your applications.

:::

:::tip Clarifai Community Platform 

[Clarifai's full stack portal](https://docs.clarifai.com/clarifai-basics/community) is the ultimate AI workspace for any skill level. It provides everything you need to build reliable AI-powered apps. With its low-code, no-code  solutions, you can easily discover, create, and share futuristic AI models, workflows, app components, and more. 


:::
