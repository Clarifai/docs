# Supported Formats

**Clarifai works with the most popular image, video, audio, and text formats**
<hr />

The API supports a wide range of the following data formats:

## Image formats

* JPEG
* PNG
* TIFF
* BMP
* WEBP
* GIF

## Video formats

* AVI
* MP4
* WMV
* MOV
* 3GPP

## Audio formats

* WAV

## Text formats

* Plain text
* CSV files
* TSV files
