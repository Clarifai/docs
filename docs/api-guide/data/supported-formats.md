# Supported Formats

**Clarifai works with the most popular image, video, and text formats**
<hr />

## The API supports the following image formats:

* JPEG
* PNG
* TIFF
* BMP
* WEBP

## The API supports the following video formats:

* AVI
* MP4
* WMV
* MOV
* GIF
* 3GPP

## The API supports the following text formats:

* Plain text
