# Datasets
**Manage the datasets for training, testing, and evaluating your models**
<hr />

A dataset is a set of data examples. With Clarifai datasets, you can manage the inputs you want to use for visual search, model training, evaluation, and more. Datasets are stored as convenient snapshots in datasets tables.
