# Integrating Clarifai With CrewAI

**Build collaborative AI agent teams with CrewAI and Clarifai**
<hr />

[CrewAI](https://github.com/joaomdmoura/crewAI) is a framework for orchestrating autonomous AI agents that work together as a crew. Each agent has a specific role, goal, and set of tools. By integrating with Clarifai, you can power your agent crews with any model on the Clarifai platform.

## Getting Started

### Installation

```bash
pip install crewai litellm
```

CrewAI uses LiteLLM for model routing, which supports Clarifai's OpenAI-compatible endpoint.

### Configure the LLM

```python
import os
from crewai import LLM

llm = LLM(
    model="openai/openai/chat-completion/models/gpt-oss-120b",
    base_url="https://api.clarifai.com/v2/ext/openai/v1",
    api_key=os.environ["CLARIFAI_PAT"]
)
```

The model ID follows the LiteLLM convention: `openai/{user_id}/{app_id}/models/{model_id}`. Convert from Clarifai URLs like this:

- `https://clarifai.com/openai/chat-completion/models/gpt-4o` → `openai/openai/chat-completion/models/gpt-4o`
- `https://clarifai.com/qwen/qwenLM/models/Qwen3_5-397B-A17B` → `openai/qwen/qwenLM/models/Qwen3_5-397B-A17B`

### Create Agents and Run a Crew

```python
from crewai import Agent, Task, Crew, Process

llm = LLM(
    model="openai/openai/chat-completion/models/gpt-oss-120b",
    base_url="https://api.clarifai.com/v2/ext/openai/v1",
    api_key=os.environ["CLARIFAI_PAT"]
)

researcher = Agent(
    role="Research Analyst",
    goal="Find and summarize key information on a given topic",
    backstory="You are an expert research analyst with deep expertise in technology trends.",
    llm=llm,
    verbose=True
)

writer = Agent(
    role="Technical Writer",
    goal="Write a clear, concise summary based on research findings",
    backstory="You are a skilled technical writer who distills complex topics into readable summaries.",
    llm=llm,
    verbose=True
)

research_task = Task(
    description="Research the latest developments in {topic}. Focus on key trends, major players, and recent breakthroughs.",
    expected_output="A detailed research summary with key findings",
    agent=researcher
)

writing_task = Task(
    description="Based on the research findings, write a concise executive summary suitable for a technical audience.",
    expected_output="A polished executive summary (300-500 words)",
    agent=writer
)

crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, writing_task],
    process=Process.sequential,
    verbose=True
)

result = crew.kickoff(inputs={"topic": "speculative decoding in LLM inference"})
print(result)
```

### Using Different Models per Agent

You can assign different Clarifai models to different agent roles:

```python
# Use a larger model for complex reasoning
reasoning_llm = LLM(
    model="openai/openai/chat-completion/models/gpt-oss-120b",
    base_url="https://api.clarifai.com/v2/ext/openai/v1",
    api_key=os.environ["CLARIFAI_PAT"]
)

# Use a smaller model for simpler tasks
fast_llm = LLM(
    model="openai/qwen/qwenLM/models/Qwen3-30B-A3B-FP8",
    base_url="https://api.clarifai.com/v2/ext/openai/v1",
    api_key=os.environ["CLARIFAI_PAT"]
)

lead_analyst = Agent(role="Lead Analyst", llm=reasoning_llm, ...)
data_collector = Agent(role="Data Collector", llm=fast_llm, ...)
```

:::info

For complete examples including tool integration and YAML-based agent configuration, visit the [Clarifai Examples Repository](https://github.com/Clarifai/examples/tree/main/agents/CrewAI).

:::

:::tip

Browse models on the [Clarifai Community](https://clarifai.com/explore) to find the right model for each agent role. Mix larger models for reasoning-heavy agents with smaller models for simpler tasks to optimize cost.

:::
