# Integrating Clarifai With DeepEval

**Evaluate and test your LLM applications with confidence**
<hr />

[DeepEval](https://docs.confident-ai.com/) is an open-source evaluation framework for LLM applications. It provides a comprehensive suite of metrics and tools to evaluate the performance, quality, and reliability of your language model applications. DeepEval makes it easy to measure hallucination, answer relevancy, contextual precision, and many other important metrics for production LLM systems.

Clarifai offers an all-encompassing platform that covers the complete AI lifecycle—from data exploration to inferencing. By integrating Clarifai with DeepEval, you can leverage Clarifai's powerful LLM models for evaluation tasks while maintaining rigorous testing standards for your AI applications.

## Why Use DeepEval with Clarifai?

Combining DeepEval's evaluation framework with Clarifai's LLM capabilities provides several advantages:

- **Comprehensive Testing**: Use DeepEval's extensive metric library to thoroughly evaluate LLM responses from Clarifai models.
- **Production-Ready Models**: Test and validate Clarifai's production-scale LLMs before deployment.
- **Customizable Evaluation**: Create custom metrics tailored to your specific use case using Clarifai models as judges.
- **Continuous Monitoring**: Integrate evaluation into your CI/CD pipeline to ensure consistent model performance.
- **Multi-Model Comparison**: Compare different Clarifai LLM models to find the best fit for your application.

## Key Evaluation Metrics

DeepEval supports various metrics that can be used with Clarifai models:

- **Answer Relevancy**: Measure how relevant generated answers are to the input questions
- **Faithfulness**: Evaluate if responses are grounded in provided context without hallucination
- **Contextual Precision**: Assess the quality and relevance of retrieved context
- **Contextual Recall**: Measure if all relevant information is retrieved
- **Toxicity**: Detect harmful or inappropriate content in model outputs
- **Bias**: Identify potential biases in model responses

## Getting Started

To integrate Clarifai with DeepEval, you'll need:

1. A Clarifai account with API access (sign up at [clarifai.com/signup](https://clarifai.com/signup))
2. DeepEval installed in your Python environment
3. Your Clarifai Personal Access Token (PAT)

The integration allows you to use Clarifai's LLM models both as the models being evaluated and as judge models for custom metrics.

## Example Use Cases

- **RAG System Evaluation**: Test retrieval-augmented generation systems using Clarifai's vector search and LLMs
- **Chatbot Quality Assessment**: Evaluate conversational AI applications for coherence, relevance, and safety
- **Content Generation Testing**: Validate the quality of generated content across different Clarifai LLM models
- **Model Comparison**: Benchmark different Clarifai models against each other for specific tasks
